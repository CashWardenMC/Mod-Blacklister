# Mod-Blacklister
A Minecraft mod that requires itself on clients and blacklists specified mods via blacklist.json.
Upon running a client or server with this on a config file, blacklist.json will be generated.
Within that config file, you can enter any mod ID that you want to blacklist.
Keep in mind this is not a very secure way to enforce disallowed mods on a server, because it can be bypassed without too much diffculty.
